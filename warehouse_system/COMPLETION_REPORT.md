## 🎯 实现的优化目标

### 目标1：✅ 最小化总完成时间 (Makespan)
**实现方式**:
- 考虑机器人当前队列长度
- 优先分配给空闲或负载低的机器人
- 权重：队列时间×35%

### 目标2：✅ 最小化总行驶距离
**实现方式**:
- 选择距离最近的机器人
- 使用A*规划最短路径
- 权重：距离×40%

### 目标3：✅ 机器人负载均衡
**实现方式**:
- 计算工作负载（任务数+负载率）
- 分配给负载最低的机器人
- 权重：工作负载×25%

### 目标4：✅ 避免拥堵/冲突
**实现方式**:
- 时间-空间碰撞检测
- 历史窗口管理（100秒）
- A*算法绕过障碍物

## 📈 关键数据结构

### Robot（机器人）
```python
{
    'id': 1,
    'x': 10, 'y': 20,                    # 位置
    'state': 'idle',                    # 状态（空闲/忙碌/移动/充电）
    'battery': 85.5,                    # 电量 0-100%
    'load': 45,                         # 当前负载
    'tasks_in_queue': 2,                # 待执行任务数
    'total_distance': 234.5,            # 总行驶距离
    'tasks_completed': 23,              # 完成任务数
}
```

### Order（订单）
```python
{
    'id': 1,
    'pickup_location': (20, 20),        # 取货点
    'delivery_location': (40, 40),      # 送货点
    'priority': 'HIGH',                 # 优先级
    'status': 'assigned',               # 状态
    'assigned_robot_id': 1,             # 分配的机器人
    'quantity': 5,                      # 数量
}
```

### Metrics（指标）
```python
{
    'total_distance': 1250.5,           # 总行驶距离
    'total_tasks_completed': 45,        # 已完成任务
    'average_battery': 82.3,            # 平均电量
    'total_assignments': 45,            # 总分配次数
}
```
